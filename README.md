# Утилита предназначения для запуска приложений и набора приложений.

## Этапы установки и первого использования:

1. Скачать архив из релизов
2. Распаковать и добавить в переменную среды путь до распакованной папки
3. Импортировать настройки для запуска приложений

   Пример файла конфигурации находится в [путь до файла](deps/programs.xml)
   Схема для формирования файла конфигурации находится в [путь до файла](deps/schema.xsd)
   Команда для импорта

   ```
   autostart --import path_to_file.xml
   ```

##  Возможности утилиты
- Импортировать настройки
  ```
  autostart --import path_to_file.xml
  ```
- Запустить приложение
  ```
  --start-application name_application
  ```
  ```
  --sa name_application
  ```
- Запустить набор приложений

  ```
  --start-project name_project
  ```

  ```
  --sp name project
  ```
- Посмотреть список приложений
  ```
  --display-application
  ```
  ```
  --da
  ```
- Посмотреть наборы приложений
  ```
  --display-projects --wa
  ```
  ```
  --da --wa
  ```
